# ChevronTileInc.

## Description

This project is a web application for a tile installation company. It allows users to view services, browse a gallery, request estimates, and search submitted requests.

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open browser:
   http://localhost:3000

## Features

- Multi-page website (7 pages)
- Styled with CSS
- Form validation using JavaScript
- Data stored in MySQL database
- Search functionality with server response
